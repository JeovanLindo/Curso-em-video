FORMATAÇÕES DE TEXTO


NEGRITO

Para usar o negrito NÃO semantico use <b>

Para usar o negrito semantico use <strong>

ITALICO

para usar itálico NÃO semantico use <i>

para usar italico semantico use <EM>

MARCA TEXTO

para usar o marca texto use <mark>

TEXTO PEQUENO

para usar o texto pequeno use <small>

TEXTO DELETADO

para usar o texto deletado use <del>

TEXTO INSERIDO

para usar o texto inserido use <ins>

TEXTO SOBRESCRITO

para usar o texto sobrescrito use <sup>

TEXTO SUBSCRITO

para usar o texto subscrito use <sub>
