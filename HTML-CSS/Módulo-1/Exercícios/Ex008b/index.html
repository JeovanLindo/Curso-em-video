FORMATAÇÕES EXTRAS


Fonte mono-espaçadas

Para usar a fonte mono-espaçada use <code>

Pré-formatado

Para usar o texto Pré-formatado use <pre>


    EXEMPLO : Lorem, ipsum dolor sit amet consectetur
    adipisicing elit. Nulla veritatis quam magnam ullam quod
    porro incidunt laudantium eaque et dolor quos, expedita
    laboriosam ea? Amet nulla quasi consectetur labore
    ducimus.

Citações

Para usar Citações use <q>

Citações completas

Para usar Citações completas use <blockquote>

EXEMPLO: Lorem ipsum dolor sit amet consectetur adipisicing elit. Eius ducimus deserunt odio magnam sunt enim officia eos repudiandae culpa omnis nostrum, ex ipsam expedita eveniet numquam, at magni error rerum!

Descrever siglas

Para descrever siglas (RJ) use <abbr>

Texto invertido

Para usar o texto invertido (texto invertido) use <bdo>
