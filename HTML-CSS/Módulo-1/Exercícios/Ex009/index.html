Listas ordenadas

Para usar as listas ordenadas use <ol>

  1. pao
  2. mostarda
  3. manteiga

Podemos usar <ol type:""> para mudar o tipo de lista por exemplo:

  1. pao
  2. mostarda
  3. manteiga
  1. pão
  2. mostarda
  3. manteiga

Listas NÃO ordenadas

Para usar as listas NÃO ordenadas use <ul>

  • pao
  • mostarda
  • manteiga

Podemos usar<ul type:""> para mudar o tipo de lista por exemplo

  • pao
  • mostarda
  • manteiga
  • pao
  • mostarda
  • manteiga

Listas Juntas

Para juntar listas basta usar <ol> ou <ul> dentro delas mesmas podendo mudar o type tambem, por exemplo :

  1. pão
    1. farinha
    2. ovo
    3. manteiga
  2. manteiga
    • ovo
      1. frito
    • leite
    • elefante

Listas de definição

Para usar a lista de definição use <dl>, depois use <dt> para botar o termo, depois use <dd> para a descrição/definição
